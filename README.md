## Goals

This repository seeks to compile all the materials for each course in the second semester of Electronics and Telecommunications studies. I hope this repository could allow you to study even if you're offline. 
## Quick Start Guide

To use this repository as intended `git clone` it to a desired folder on your machine and proceed to open it as a vault using [Obsidian](https://obsidian.md/). If you're not yet using [Obsidian](https://obsidian.md/), you can download it here: 

https://obsidian.md/download
